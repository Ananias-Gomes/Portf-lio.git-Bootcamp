Bem-vindo ao repositório do meu Portfólio Profissional (Versão 1.0). Este projeto foi desenvolvido como parte do Desafio de Versionamento e integração Contínua (Git + GitHub).

## 🚀 Sobre o Projeto
Este site é o meu cartão de visitas digital. Ele reflete meu mindset de alto rendimento, minha base em Engenharia de Software e meus objetivos focados em Inteligência Artificial. O design foi concebido com uma estética minimalista, dark e futurista, utilizando um layout de tela dividida (Split Screen) responsivo.

## 🛠️ Tecnologias Utilizadas
Este projeto foi construído "do zero", focando nos fundamentos da web para garantir performance máxima e domínio estrutural:
* **HTML5:** Semântica estrutural (`<aside>`, `<main>`, `<section>`).
* **CSS3:** Estilização, variáveis de raiz, CSS Grid, Flexbox e animações `@keyframes` (efeito fluido).
* **Git & GitHub:** Versionamento de código, ramificação e deploy.

## 📁 Estrutura do Repositório
Para manter a organização exigida, este repositório central atua como o Hub do meu código:
- `/projetos`: Projetos em desenvolvimento (App Alter, Jogo de Shogi, Site Pessoal).
- `/faculdade`: Acervo de matérias (Banco de Dados, Eng. de Software, Lógica de Programação).
